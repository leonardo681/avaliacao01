O QUE É HTML?

HTML, ou HyperText Markup Language, é uma linguagem de marcação utilizada
para definir a estrutura de um conteúdo na web, não é uma linguagem de
programação. Consiste em uma série de elementos, delimitados por tags, que
organizam e formatam o conteúdo. Cada elemento é composto por uma tag de
abertura, uma tag de fechamento e o conteúdo em si. Por exemplo, a tag "p" inicia
um parágrafo, enquanto "/p" indica seu fim. Um erro comum entre iniciantes é
esquecer a tag de fechamento, o que pode resultar em exibições inesperadas.
Além dos elementos básicos, HTML permite o uso de atributos, que são
especificações adicionais que ajustam o comportamento ou a aparência dos
elementos. Para serem corretamente aplicados, os atributos devem ter um espaço
entre eles e o nome do elemento, seguido de um sinal de igual e aspas que
delimitam seu valor.
Existem também elementos vazios, como "img", que não possuem conteúdo
interno e não requerem tag de fechamento.

O documento HTML começa com a declaração "!DOCTYPE html", que assegura
a conformidade com as normas da linguagem. O elemento "html" envolve todo o
conteúdo da página, enquanto o "head" contém informações não exibidas

diretamente, como metadados e links para CSS. O elemento "meta charset="utf-
8" define o conjunto de caracteres, garantindo a compatibilidade com diversos

idiomas. O "title" define o título que aparece na aba do navegador, e o"body"
contém todo o conteúdo visível ao usuário, como textos, imagens e vídeos. Essas
tags são fundamentais para a criação e apresentação de páginas na web."/p"
